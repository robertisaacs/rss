# AI-Augmented RSS Feed Reader

A personalized, algorithm-free content consumption tool with AI-assisted summarization and tagging.

## Features

- On-demand RSS feed fetching and parsing
- Local storage of articles, metadata, and summaries
- AI-powered article summarization
- Automatic topic tagging
- Semantic search via embeddings

## Technology Stack

- **Framework**: Next.js (React)
- **RSS Library**: Feedparser (Node.js)
- **Database**: SQLite (via Prisma ORM)
- **AI Integration**: OpenAI-compatible API endpoints (e.g., GPT-4 or GPT-4o)
- **Styling**: Tailwind CSS

## Getting Started

### Prerequisites

- Node.js 18+ and npm
- OpenAI API key or compatible API endpoint

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ai-rss-reader.git
   cd ai-rss-reader
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Copy the environment variables file and update it with your settings:
   ```bash
   cp .env.example .env
   ```

4. Update the `.env` file with your OpenAI API key and other settings.

5. Initialize the database:
   ```bash
   npx prisma migrate dev --name init
   ```

6. Start the development server:
   ```bash
   npm run dev
   ```

7. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

1. Add RSS feeds via the "Manage Feeds" page
2. View articles from all feeds on the home page
3. Use semantic search to find articles by meaning, not just keywords
4. Browse articles by tags that are automatically generated by AI

## Database Schema

- **feeds**: Stores RSS feed information
- **articles**: Stores article content, summaries, and embeddings
- **tags**: Stores topic tags
- **article_tags**: Junction table for article-tag relationships
